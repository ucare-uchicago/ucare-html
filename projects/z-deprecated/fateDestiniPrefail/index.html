About and Contacts

In this page, you can download our failure testing framework for distributed systems written in Java. Since this is a research prototype, we will be adding code, READMEs, and comments only by demand.

For documentation, please read: Fate and Destini and PreFail

Please download the test programs first. Then please choose one of the target systems that you would like to test (e.g. HDFS).

Download Test Program + PreFail

Download these two files and untar them in your chosen project directory (e.g. "test/")

  • test-prog-0.2.tar.gz
  • test-workload-0.2.tar.gz

Then please read the test-prog's README file (test/test-prog/README.txt) followed by the test-workload's README file (test/test-workload/README.txt).

Since this is a research prototype, we are adding code, READMEs, and comments only by demand. If you want to keep yourself up-to-date, you can checkout the svn repositories below.

  • svn co https://prefail.svn.sourceforge.net/svnroot/prefail/test-prog/trunk
  • svn co https://prefail.svn.sourceforge.net/svnroot/prefail/test-workload/trunk

Download HDFS + PreFail

Downloading HDFS that has been integrated with PreFail is as similar as the above steps. Please download these two files, untar them in a directory (e.g. "hdfs-prefail/"). Note that they don't have to reside in the same directory. This is just for simplicity of the instruction.

  • hdfs-prefail-system-0.2.tar.gz
  • hdfs-prefail-workload-0.2.tar.gz

Then please read the system's README file (hdfs-prefail-system/README.txt) followed by the workload's README file (hdfs-prefail-workload/README.txt).

Finally, keep yourself up-to-date via these svns:

  • svn co https://prefail.svn.sourceforge.net/svnroot/prefail/hdfs-prefail-system/trunk
  • svn co https://prefail.svn.sourceforge.net/svnroot/prefail/hdfs-prefail-workload/trunk

Download Cassandra + Fate

Downloading Cassandra that has been integrated with Fate is similar to the above steps. Please download these two files, untar them in a directory(e.g. "cass-fate/"). Note that they don't have to reside in the same directory. This is just for simplicity of the instruction.

  • cass-fate-system-0.1.tar.gz
  • cass-fate-workload-0.1.tar.gz

Then please read the cass-fate-system's README files (cass-fate-system/FI-README.txt) followed by the workload's README file (cass-fate-workload/README.txt).

Finally, keep yourself up-to-date via git: cass-fate's github page
